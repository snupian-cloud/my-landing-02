# MIDAS AI 아카데미 - AI 융합 전문 강사 프로필 사이트

## 🎯 프로젝트 소개

**MIDAS AI 아카데미**는 25년 교육 사업 경력을 보유한 AI 융합 전문가 **김진성(Jinsung Kim)**님의 브랜딩 및 강의 홍보를 위한 전문 프로필 웹사이트입니다.

### 주요 특징
- 🎨 **모던한 AI 테마 디자인**: 그라데이션과 애니메이션을 활용한 현대적인 디자인
- 💼 **25년 교육 사업 노하우**: 교육 프랜차이즈 운영 경험 + AI 기술 융합
- 📚 **레퍼런스 관리 시스템**: 강의, 프로젝트, 컨설팅 실적 체계적 관리
- ⭐ **실시간 수강 후기**: 평점 및 리뷰 시스템
- 📨 **문의 관리 시스템**: 강의 문의 접수 및 관리
- 📱 **완전 반응형 디자인**: 모바일, 태블릿, 데스크톱 최적화

## 🚀 현재 구현된 기능

### ✅ 완료된 기능

#### 1. 메인 페이지 (index.html)
- **AI 테마 그라데이션 배너**: 애니메이션 효과가 적용된 모던한 히어로 섹션
- **히어로 섹션**: 25년 교육 경력, 효율성 300% 증대, 시니어 맞춤 교육 강조
- **프로필 섹션**: 
  - 전문 프로필 사진 (체크무늬 재킷, 안경 착용)
  - 학력(서울대학교), 자격(AI 작가, AI 강사)
  - 경력 타임라인 (2009-현재)
- **AI 전문 강의 영역**:
  - ChatGPT 실전 활용
  - 이미지 생성 AI (Midjourney, DALL-E)
  - AI 영상 제작
  - AI 마케팅 자동화
  - 인플루언서 커머스
  - 시니어 맞춤 AI 교육
- **차별화 포인트**: 검증된 실전 경험, 시니어 눈높이 교육, 돈이 되는 AI, 1:1 맞춤 컨설팅
- **네비게이션**: 부드러운 스크롤, 모바일 메뉴 지원

#### 2. 레퍼런스 시스템
- **카테고리 필터링**: 전체 / 강의 / 컨설팅 / 프로젝트 분류
- **레퍼런스 카드**: 이미지, 클라이언트, 기술 스택, 기간 표시
- **데이터베이스 연동**: RESTful API를 통한 동적 데이터 로딩
- **샘플 데이터**: 8개의 실전 레퍼런스 사전 등록
  - 서울시니어센터 ChatGPT 실전 활용 교육
  - 네이버 크리에이터 아카데미 AI 콘텐츠 제작 워크샵
  - 블루래빗 브랜드 공동구매 AI 효율화 프로젝트
  - 서울시 소상공인지원센터 AI 마케팅 컨설팅
  - 패스트캠퍼스 AI 이미지 생성 마스터 클래스
  - 브레이닝 교육 콘텐츠 AI 리뉴얼
  - 콘텐츠 크리에이터 협회 AI 영상 편집 교육
  - 대기업 교육센터 기업 임직원 AI 리터러시 교육

#### 3. 리뷰 시스템
- **전체 평점 표시**: 평균 평점 및 총 리뷰 수 자동 계산
- **별점 시스템**: 5점 만점 시각화
- **리뷰 작성 모달**: 이름, 회사, 강의명, 평점, 후기 입력
- **실시간 업데이트**: 리뷰 작성 즉시 반영
- **샘플 리뷰**: 6개의 수강생 후기 사전 등록
  - 평균 평점: 4.8/5.0
  - 시니어, 인플루언서, 소상공인, 기업 임직원 등 다양한 수강생 후기

#### 4. 문의 관리 시스템
- **문의 폼**: 이름, 이메일, 연락처, 회사명/소속, 제목, 내용
- **문의 상태 관리**: pending / in_progress / completed
- **연락처 정보**: 이메일, 블로그, 전문 분야, 상담시간
- **소셜 미디어 링크**: 블로그, 이메일

#### 5. 데이터베이스 스키마

##### `inquiries` (문의)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 식별자 |
| name | text | 문의자 이름 |
| email | text | 이메일 |
| phone | text | 연락처 |
| company | text | 회사명/소속 |
| subject | text | 문의 제목 |
| message | text | 문의 내용 |
| status | text | 처리 상태 (pending/in_progress/completed) |
| created_at | datetime | 문의 일시 |

##### `reviews` (리뷰)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 식별자 |
| name | text | 수강생 이름 |
| company | text | 회사명/소속 |
| course | text | 수강한 강의 |
| rating | number | 평점 (1-5) |
| comment | text | 리뷰 내용 |
| is_featured | bool | 추천 리뷰 여부 |
| created_at | datetime | 작성 일시 |

##### `references` (레퍼런스)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 식별자 |
| title | text | 프로젝트/강의 제목 |
| category | text | 카테고리 (강의/프로젝트/컨설팅) |
| client | text | 클라이언트/기관명 |
| description | text | 상세 설명 |
| tech_stack | array | 사용 기술 스택 |
| duration | text | 기간 |
| image_url | text | 이미지 URL |
| link_url | text | 관련 링크 |
| is_featured | bool | 대표 레퍼런스 여부 |
| created_at | datetime | 등록 일시 |

```javascript

  "duration": "2024.01 - 현재",

  "link_url": "",
```

```javascript
// 리뷰 목록 조회
```

## 🎨 주요 기능 상세

### 1. AI 테마 그라데이션 배너
- **위치**: 히어로 섹션 배경
- **효과**: 보라색-파란색 그라데이션 + 라디얼 그라데이션 애니메이션
- **반응형**: 모든 화면 크기 지원
- **애니메이션**: 10초 주기 opacity 변화

### 2. 동적 레퍼런스 필터링
- **카테고리 버튼**: 클릭 시 즉시 필터링
- **애니메이션**: 부드러운 전환 효과
- **카운트**: 카테고리별 레퍼런스 수 자동 계산

### 3. 리뷰 평점 시스템
- **평균 평점**: 자동 계산 및 표시
- **별점 입력**: 클릭 및 호버 인터랙션
- **실시간 업데이트**: 리뷰 작성 즉시 반영

### 4. 경력 타임라인
- **시각적 표현**: 세로 타임라인 + 동그라미 마커
- **그라데이션 라인**: 파란색-보라색 그라데이션
- **상세 정보**: 기간, 역할, 주요 성과

### 5. 반응형 디자인
- **데스크톱** (1024px+): 3컬럼 레이아웃, 사이드바 프로필
- **태블릿** (768px-1023px): 2컬럼 레이아웃
- **모바일** (767px 이하): 1컬럼 레이아웃, 햄버거 메뉴

## 📝 사용 가이드

### 레퍼런스 추가하기
```javascript
// JavaScript 콘솔 또는 관리자 페이지에서 실행
const newReference = {
  title: "새로운 강의 제목",
  category: "강의",
  client: "기관명",
  description: "강의 설명",
  tech_stack: ["ChatGPT", "Midjourney", "AI 마케팅"],
  duration: "2024.01 - 현재",
  image_url: "https://images.unsplash.com/...",
  link_url: "",
  is_featured: false,
  created_at: Date.now()
};

await fetch('tables/references', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify(newReference)
});
```

### 문의 상태 변경하기
```javascript
// 문의 ID로 상태 업데이트
await fetch('tables/inquiries/{inquiry_id}', {
  method: 'PATCH',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({ status: 'completed' })
});
```
